
# E-Commerce-App

Follow these instructions to set up and run the project on your local machine.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Nagesh0137/E-Commerce-App.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
